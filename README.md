<h2>运行环境：</h2>
使用python3运行<br/>
<h2>支持的功能：</h2>
  添加后门账户<br/>
  为账户添加sudo权限<br/>
  隐藏bash命令<br/>
  软链sshd后门<br/>
  crontab反向Shell<br/>
  写入公钥<br/>
  持久化反向Shell<br/>
